## 📝 使用指南

### 快速开始

查看 `QUICKSTART.md` 获取5分钟快速启动指南。

### 开发

```bash
# 后端
cd backend
pip install -r requirements.txt
python init_project.py
python main.py

# 前端
cd frontend
npm install
npm run dev
```

### 测试账号

使用 `init_project.py` 生成的邀请码注册新账号。
第一个注册的用户自动成为管理员。
